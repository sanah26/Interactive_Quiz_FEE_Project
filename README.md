# 🚀 CosmoQuiz

CosmoQuiz is an interactive space-themed online quiz web application developed using HTML, CSS, and JavaScript.

## 📌 Project Overview

This project allows users to sign up, log in, select a difficulty level, attempt quiz questions, and view their final score, accuracy, high score, and leaderboard.

## ✨ Features

- User Sign-Up and Login
- Forgot Password option
- Difficulty levels: Easy, Medium, Hard
- Random quiz questions
- Timer and progress bar
- Correct/Wrong answer popup
- Explanation with image support
- Final score and accuracy
- High score tracking
- Leaderboard using LocalStorage
- Responsive space-themed UI

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- LocalStorage
- SessionStorage

## 📂 Project Pages

- Login Page
- Sign-Up Page
- Forgot Password Page
- Home Page
- Quiz Page
- Result Page

## 💾 Storage Usage

- LocalStorage is used to store user details, high score, and leaderboard.
- SessionStorage is used to store current username, score, total questions, and selected difficulty.

## 🎯 Purpose

The purpose of this project is to create an engaging and user-friendly quiz platform while demonstrating front-end development concepts such as DOM manipulation, event handling, functions, arrays, and browser storage.

## 🚀 How to Run

1. Download or clone the repository.
2. Open `LOGIN_PAGE_@.html` in a browser.
3. Create an account using Sign-Up.
4. Login and start the quiz.
